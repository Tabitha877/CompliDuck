# CompliDuck

## About
CompliDuck is a webpage with random duck images and heartfelt compliments to cheer you up!

## installation
+ Clone the repo 
+ Go to your code edditor
+ cd into the folder
+ Install dependencies
  `npm i`
+ Run the code with nodemon / node
  `nodemon index.js`
  `node index.js`
+ Type in your google search bar ***localhost:3000***

## Used npm package
[cheer-me-up package](https://www.npmjs.com/package/cheer-me-up)

## Used API
[Random-d.uck](https://random-d.uk/api)
